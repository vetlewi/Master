Detailed Description

Formatted stream output.

Mapping of detector objects.

This is a class that reads a detector array from file and creates a mapping based on the ADC address of the detector.

bool Detectors::setAll ( std::string  filename)

Set one or more detector from external text file.

The format for the text file is: 

<adc-address> (tab/space(s)) <detector-name> (tab/space(s)) <detector-type>

Only one detector for each line.
adc-address should be an unsigned integer.
detector-name should be a string without spaces.
The detector-type should be given as:
gamma-detectors: "gamma" or "GAMMA" (case insensitive)
silicon-detectors: "silicon" or "SILICON" (case insensitive)
Constant fraction discriminator: "cfd" or "CFD" (case insensitve)
None: "none" or "NONE" (case insensitive)
Unknown: If not any of the above. Comment lines starts with "#". The function will print a message if an error was encounted.

Returns
True if the file was parsed completely; if false, the file might be wrongly formatted and some detectors might not have been mapped correctly.
Parameters
filenameThe filename of the file to read.
