Detailed Description

Buffer type used in this project.

Subclass of WordBuffer with a spesific size.

This class has a predetermined size equal to half of the buffer size used in the data files used in this project. This is beacuse half of the buffer is the CFD reading which is useless for the sorting and is discarded.
